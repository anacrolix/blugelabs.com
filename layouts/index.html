Welcome to Bluge Labs, an organization focused on supporting high-quality search software for Go. Bluge Labs is made possible by our sponsors. These supporters recognize the value of these projects, and work together with Bluge Labs to ensure their future. Our top priorities:
  • Adopt Go Modules, including refactoring larger packages into multiple modules where appropriate.
  • Begin following a release train model, allow users to build software around stable versions with known support time tables.
Keep reading to learn more about the major projects we support.

Bleve

Bleve is the core project at the heart of Bluge Labs today. Started in 2014 at Couchbase, Bleve has come a long way. Today there are many companies shipping products that rely on Bleve indexing and search. Bluge Labs' role is to ensure that Bleve remains a healthy open-source project with regular releases.

Vellum

Vellum is a library implementing a data-structure known as a Finite State Transducer. Today it's primarily used internally by the Bleve code base. However, there are projects using Vellum directly, and we believe it's full potential has not been realized. Here at Bluge Labs, we plan to continue to support Vellum, and encourage wider adoption of this technology.
